# 02 — Requisitos Funcionais (FR)

## Lista de FR
- **FR-01** Criar post (texto + timestamp + author_id).
- **FR-02** Obter post por ID.
- **FR-03** Seguir/deixar de seguir usuário.
- **FR-04** Listar seguindo (paginado).
- **FR-05** Listar seguidores (paginado).
- **FR-06** Ler feed (paginado por cursor, ordenado por tempo).
- **FR-07** Fan-out: `PostCreated` atualiza feeds para usuários não celebridade.
- **FR-08** Estratégia híbrida: posts de celebridades são mesclados no read.

## Observações
- Todas as operações devem respeitar **autoridade vs derivação**.
- O feed é **somente texto** e ordenado por tempo com desempate por `post_id`.
